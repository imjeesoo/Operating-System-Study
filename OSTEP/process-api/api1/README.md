1. api1.c : fork()를 이용한 부모와 자식 프로세스의 독립적인 주소 공간 확인
1) 실험 목적
fork()를 통해 생성된 자식 프로세스가 부모의 주소 공간을 공유하는지, 아니면 독립적인 공간을 가지는지 확인한다.
2) 구현 세부 사항 (api1.c)
정수형 변수 x를 100으로 초기화한 후 fork()를 호출한다.
자식 프로세스: 자신의 x 값을 200으로 수정한다.
부모 프로세스: wait(NULL)로 자식의 종료를 기다린 후, 자신의 x 값을 300으로 수정한다.

자식 프로세스에서 x를 변경해도 부모의 x 값(100)에는 영향을 주지 않았다.
이는 fork()시 자식 프로세스를 위해 별도의 주소 공간이 생성됨을 의미하며, 
운영체제의 핵심인 프로세스 격리(Process Isolation) 원리를 보여준다.
